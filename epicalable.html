We Know Design

Our startup aims to make our apps more vibrant, sustainable and efficient.
These days whenever a product is being created, developers need to not only think about the backend code but also now need to think about integrating the backend code to the frontend UI/UX buttons and funtions so that the users can navigate through the product using the frontend without prior knowledge or understanding of the Backend coding.
So it is important for the Frontend design to be understandable and fool-proof for the user and as for the developers, design helps us to easily identify and correct issues as it helps us to quickly locate sections of unstable code using visual cues typically found in our documentations and codes.

OUR PROJECTS

What Our organisation has done so far

Our Organisation uses Visual Studios Code to code and Git for version control for most of our projects and endeavours. While we want to create successful products we would need to start somewhere to show the world that we are capable of creating, testing and deploying products and also show that we have the ability to also market our products to our indented targeted audience. We also place our product's code in Github for our coding audience to see and also help contribute ideas and test it to find bugs for us to build a better version of our product.

JARVIS
Started in 2020 this Chatbot built using python is capable of chatting and retrieving information, daily news and other data from the internet for it's user. It's bundled with a sleek FrontEnd for better User Experience (UX) currently being hosted and stored in github. This project is our First Flagship Project and has gotten 60 stars on github (at the time of writing this) so far.

ComDash
Announced in 2022 as Epicalable's most ambitious project after Jarvis, ComDash is an All-in-one Web-based Dashboard for News, Weathers and some more cool features coming soon. This project is our first time working with JavaScript, It's current emphasis is on utilizing coding to create aesthetic web-based dashboards for a greater UX.

Stay tuned to the latest updates by 'starring' and 'watching' the repositories over at GitHub.
